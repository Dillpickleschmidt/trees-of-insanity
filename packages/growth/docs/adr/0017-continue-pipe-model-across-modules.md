# Continue the pipe model across module attachments

Treat parent-terminal/child-root attachments as edges in one continuous basipetal Eq. 8 diameter calculation. A child's currently developed root diameter supplies its parent terminal and contributes to upstream parent diameters, preventing visible discontinuities and matching the paper's attachment rule. Its eventual target diameter must not contribute before it develops. Shedding removes that contribution. Prototype topology remains precomputed while dynamic diameter values traverse attachments.
